A selection of some of the more interesting projects that I have worked on recently:

PrettyLittleThing

Writing AWS lambda functions in nodeJs to deliver a micro-services based architecture. Also building a product information management system using Symfony.

eBay

Building middleware with PHP to integrate order management systems for retailers such as Mothercare with different warehousing systems.

Carphone Warehouse

Working with a high street mobile phone retailer to build a new checkout system with Symfony for use across several brands in the group.

Plusnet

Working with a well known telecoms provider to adopt new Ofcom rules and fix integration issues with their billing engine written in PHP and Java.

Sky

Consolidating a number of legacy systems to produce a single API resource using Symfony for use across Sky websites and Sky Sports mobile apps.

Reading Room

Working with a large design agency to build a community-based website for the British Cycling Team using Laravel.
